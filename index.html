Stock.io — Real-Time Stock Recs

Streamlit app that combines real-time news sentiment (web-scraped last 3 days) with an LSTM forecast on historical OHLC to produce dual recommendations.

StreamlitWeb ScrapingNLPLSTM
Read More

System

  • Scrapes finance headlines for a chosen ticker; aggregates a sentiment score (rolling 3 days).
  • LSTM predicts intraday range from historical OHLC; outputs trend + confidence.
  • UI blends both signals into a clear “dual recommendation”.

Consulting value

Shows how to combine unstructured (news) & structured (time series) data into one decision app.
